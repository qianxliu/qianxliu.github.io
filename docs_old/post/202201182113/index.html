平面n边形组成封闭立体几何图形，所需的最小n边形数量

前言

‘Find a ...’ or ‘Find all ...’ questions, which requires one to find something (or everything) that satisfies certain requirements; 首先，修改问题为：平面x边形组成封闭y面体，所需的最小x边形数量

Objective

  • yy面体
  • xx边形

Notation

  • xx表示多边形边数，
  • yy表示yy面体，
  • zz表示yy面体顶点个数。
  • aa表示每点平均共享边条数。

Equation

  • 自然约束：xx边形组成yy面体有xy2\frac{xy}{2}条边，zz个点，x≥3,y≥4,z≥4x \ge 3,y \ge 4,z \ge 4
  • yy面体每点平均被aa条边共享，即z=xyaz=\frac{xy}{a},a≥3a \ge 3
  • 欧拉公式：z+y−xy2=2z+y-\frac{xy}{2}=2

目标：确定x的取值范围

通过欧拉公式加自然约束和yy面体每点平均被aa条边共享，可得y(1−a−22ax)=2y(1-\frac{a-2}{2a}x)=2。 可见，当a−22ax≥3(a−2)2a≥1\frac{a-2}{2a}x \ge \frac{3(a-2)}{2a} \ge 1时，欧拉方程不成立，即a≥6时方程不成立。a \ge 6时方程不成立。

若问题为平面6边形可以组成y面体，可用反证法

若存在只由6边形组成立体图形，由欧拉公式有2x+x−3x=22x+x-3x=2，不成立

结论

平面xx边形组成yy面体，所需的最小xx边形数量为3，4，53，4，5。
