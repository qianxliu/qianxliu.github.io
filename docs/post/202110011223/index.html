线性代数Ax=λxAx=λx

这篇文章主要讲考研数学的重点之一，也是线性代数（数学专业中这一部分会并入高等代数中，实际上线性代数是对这一部分最精确的描述，如MIT等大学采用的方式）中关于“变化(change)”的一部分。微分即是对连续函数变化的讨论。 Ax=λxAx=λx 是基本公式。 行列式∣A−λI∣|A-λI| 采用考研方式，原文是det(A−λI)det(A-λI) ，后不赘述。

特征值简介（基础）

Ax=λxAx=λx 意为特征向量xx 在乘AA 后时保持相同的方向。而λλ 特征值为特定向量xx 的变化率。

  • 特征向量 xx 与 AxAx 位于同一条线上：Ax=λxAx=λx 。特征值是λλ
  • 如果 Ax=λxAx=λx 那么 A2x=λ2xA^2x=λ^2x 并且 A−1x=λ−1xA^{-1}x=λ^{-1}x 且(A+cI)x=(λ+c)x(A+cI)x=(λ+c)x ：xx 是相同的
  • 如果 Ax=λxAx=λx 那么 (A−λI)x=0(A-λI)x=0 且 A−λIA-λI 是奇异的，并且∣A−λI∣=0|A-λI|=0 。nn 个特征值。
  • 通过∣A∣=(λ1)(λ2)...(λ3)|A|=(λ_1)(λ_2)...(λ_3) 和对角线和Σann=Σλ\Sigma{a_{nn}}=\Sigma{λ} 检查特征值λλ
  • 投影矩阵的λ=1和0λ=1和0 。反射矩阵的λ=1和−1λ=1和-1 。旋转矩阵的λ=eiθ和e−iθλ=e^{iθ}和e^{-iθ} ：复数！

我们想要当 λλ 乘以 AA 时不会改变方向的特征向量 xx 。 几乎所有的向量在乘以AA 的时候都会更改方向，但某些异常向量 xx 与 AxAx 方向相同。这些是特征向量。 而特征值λλ 告诉我们特征向量的状态（变化率或不变），当其为0时，意味特征向量xx 在零空间。

注意：特征向量是不唯一但是方向唯一的。而AA 列向量是由AA 的特征向量组合而成的。解决诸如A100A^{100} 之类的变化问题使用的是特征值。因为特征值是特定向量的变化率，所以当特征值为0到1之间时，通过A100A^{100} 的变换，这个特征值将会变成一个很小的接近于0的向量。 附录：Markov矩阵指的是最大特征值为1的矩阵，这种矩阵n(n→+∞)n(n\to+\infty) 次幂AnA^n 必将达到稳定态（原因如上），Google使用Markov矩阵指导用户搜索。

计算n阶矩阵的特征值

当且仅当A−λIA-λI 奇异时，数字λλ 是AA 的特征值。

  1. 计算A−λIA-λI 的行列式。沿对角线减去λ，该行列式以λnλ^n 或−λn-λ^n 开头。这是一个λλ 的nn 次多项式。
  2. 求解∣A−λI∣|A-λI| 以找到多项式的根。这nn 个根是AA 的nn 个特征值。它们使A−λIA-λI 奇异。
  3. 对于每个特征值λλ ，求解(A−λI)x=0(A-λI)x=0 以找到特征向量xx 。

行列式和迹

  • nn 个特征值的积等于行列式
  • nn 个特征值的和等于n个对角线元素的总和
  • A的迹=Σann=Σλ=\Sigma{a_{nn}}=\Sigma{λ}

矩阵对角化（AX=XΛAX=XΛ ）（基础，考点：对角矩阵，相似矩阵与相似对角化）

  • AX=XΛAX=XΛ 的列即Axk=λkxkAx_k=λ_kx_k 。特征矩阵ΛΛ 是对角化的。
  • XX 中的 nn 个独立特征向量对角化 AA , A=XΛX−1且Λ=X−1AXA=XΛX^{-1}且Λ=X^{-1}AX
  • 特征向量矩阵 XX 也对角化所有的幂 AkA^k ： Ak=XΛkX−1A^k=XΛ^kX^{-1}
  • 用uk=Aku0=XΛkX−1u0=c1(λ1)kx1+...+cn(λn)kxnu^k=A^ku_0=XΛ^kX^{-1}u_0=c_1(λ_1)^kx_1+...+c_n(λ_n)^kx_n 求解uk+1=Auku_{k+1}=Au_{k}
  • 没有相等的特征值=XX 可逆且AA 能够对角化；存在相等的特征值=AA 可能有太少的线性无关(independent)特征向量。那么X−1X^{-1} 不存在。
  • 每个矩阵C=B−1ABC=B^{-1}AB 都有与A相同的特征值。这些CC 与A相似。

注：ΛΛ 是λλ 的希腊文大写形式，意为λλ 存在于特征(值)矩阵ΛΛ 的对角线上。

相似矩阵

所有矩阵A=BCB−1A=BCB^{-1} 都是相似的。它们都共享 CC 的特征值。但是，不是具有相同的特征值的矩阵都是相似矩阵。 如果两个矩阵相似，则它们具有相同的特征值和相同数量的独立特征向量（但可能不相同的特征向量）。 当我们对角化 AA 时，我们会找到一个与 AA 相似的对角矩阵（特征矩阵） ΛΛ 。 可见我们是利用相似矩阵的性质，然后将B、CB、C 分别换成特征向量矩阵XX 和特征矩阵ΛΛ 来进行矩阵对角化的，也就是相似对角化。

结论

  • 所有存在n个不重复特征值的矩阵都能对角化。
  • 可逆指的是特征值不为0；对角化指的是特征向量（对XX 来说太少或太多）。两者没有关系。
  • 对应于n个不同特征值的特征向量x1,...,xjx_1, ..., x_j 应是线性无关的，才能对角化A。所有存在n个不重复特征值的矩阵都能对角化。

实对称矩阵（正交，二次型都是实对称矩阵）

写本篇时作者没有带考研书籍，可能不切合考研重点。 实对称矩阵(Symmetric Matrix)SS 有nn 个实特征值λiλ_i 以及nn 个归一化正交的特征向量q1,...,qnq_1,...,q_n . 实对称矩阵是理论和应用中最重要的矩阵，它是一种特殊的与对角矩阵相似的矩阵。

  • 实对称矩阵(Symmetric Matrix)只有实特征值。
  • 实对称矩阵的特征向量可以归一化正交。
  • 所有实特征矩阵都能对角化：S=QΛQ−1=QΛQTS=QΛQ^{-1}=QΛQ^T ，注意因为实特征矩阵的特征向量归一化正交，也可以S=QΛQTS=QΛQ^T 。
  • SS 的正特征值数等于正枢轴(pivot，化成行阶梯行最简式可得)数。二次型的正惯性系数和负惯性系数即特征值的正负个数，因为相似矩阵的性质。 因此有S=QΛQ−1=QΛQT=λ1q1q1T+...+λnqnqnTS=QΛQ^{-1}=QΛQ^T=λ_1q_1q_1^T+...+λ_nq_nq_n^T 。
