尾递归

尾递归是带着一个“迭代变量”不断向深处递归，相当于边递归，边计算，等递归完成时，结果也计算出来了。

In computer science, a tail call is a subroutine call performed as the final action of a procedure. If the target of a tail is the same subroutine, the subroutine is said to be tail-recursive, which is a special case of direct recursion. Tail recursion (or tail-end recursion) is particularly useful, and often easy to handle in implementations. 尾递归是返回递归关系最后一个动作的递归方式。

而递归中不计算任何值，到最后Return时递归求解。

## Recursion Fibonacci Numbers
def Rfib(i):
    if(i == 1 or i == 2):
        return 1
    else:
        return Rfib(i-1)+Rfib(i-2)
